# 장르: 뱀서류

# 코어 동작
  - 맵 내 적 섬멸.
  - 섬멸시 다음 Stage 진행.
  - 특정 아이템 획득시 클리어에 도움.
  - 쓰러뜨린 적 + Stage 를 통한 Score 환산

# 컨셉 정의

## 필드
- Stage 별 필드 모양 다름 -> 벽 위치, 나무 위치 등
- 기본적으로 필드는 보여지는 화면보다 큼 (실 데이터 크기 대비 Rendering 은 작음)
- 맵 바깥은 못 나가도록 고정? -> 할 수 있다면 0->Length-1 로 구현해보기. (무한)
- 아이템이 흩뿌려짐 (랜덤) -> 최대 3개만
- 필드 크기 100x100
- 랜더링 크기 20x20?

## 플레이어
- 사용자가 움직일 수 있는 객체
- 자동으로 Bullet 발사 (동서남북만 가능) -> 대각선은 추후 진행
- 사정거리; 4 / 7
- 기본 데미지 1 / 최대 5
- 기본 체력 1 / 최대 5
- **아래는 Optional**
  - 마나 게이지 5
  - 마나 풀일 경우 특정 키를 통한 스킬 사용; 사정거리 * 2 만큼 모든 적에게 데미지 * 2 만큼 주기.

## 아이템
- 보물상자를 획득하면 아래 아이템 중 랜덤으로 2가지 발생
  - 발생된 2가지 중 하나를 선택.
- 데미지 증가
- 최대 체력 증가
- 체력 회복
- 사정거리 증가
- **아래는 Optional**
  - 마나 게이지 회복 증가

## 적
- 플레이어를 찾아 플레이어에게 돌진
  - 매번 찾는 것은 어려울 듯 한데 어떻게 구현할지 생각해보기.
- 시간 혹은 스테이지 별로 돌진 속도 증가 -> 가능 여부 확인.
- 시간 혹은 스테이지 별로 체력 증가
- 시간 혹은 스테이지 별로 출현 개체수 증가
- 초기 적 체력 : 1

- 필드 랜더링 width 만큼
